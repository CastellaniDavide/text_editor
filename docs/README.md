# Description
This is the text_editor for cpp files.

## Required
 - a cpp compiler (eg. gcc for Linux)
 

## Directories structure
 - .github
   - ISSUE_TEMPLATE
     - bug_report.md
     - feature-request.md
 - bin
	 - text_editor.cpp
 - doc
   - LICENSE
   - README.md
 - log
	 - trace.log
   
### Execution examples
 - gcc text_editor.cpp -o text_editor && ./text_editor
